# Zepto-Sales-Analysis-Dashboard

Collected, organized, and analyzed Zepto's operational and marketing data using Microsoft Excel to uncover key business insights
Designed and developed a Power BI dashboard covering Customers, Inventory, Feedback, Sales, and Marketing for end-to-end performance tracking.
Created data models and DAX measures to monitor KPIs such as order volume, customer retention, revenue growth, and marketing ROI.
